# Franky — 船匠（Repurpose Agent）

將長篇文章或逐字稿改寫為不同平台格式的內容：部落格文章、SEO 優化版、社群貼文、IG Carousel。

**排程：** 手動觸發  
**狀態：** 🚧 待開發

---

## 計畫功能

- 輸入：文章草稿或影片逐字稿
- 輸出：
  - WordPress 部落格文章（SEO 優化）
  - Facebook / LinkedIn 貼文
  - Instagram Carousel 腳本（逐張文字）
- 套用 `config/style-profile.json` 的 Owner 寫作風格

## 執行

```bash
python -m agents.franky
```
