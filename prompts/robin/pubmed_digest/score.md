# 評分哲學（嚴格執行）

你的評分必須：

- **嚴格但公平**：多數好論文應落在 **3 分**，4+ 僅給真正出色的；**5 僅給十年一見的突破**
- **誠實點出弱點**：Red Flags 欄不能空白；真沒明顯問題就寫「此 abstract 未揭露明顯弱點」
- **區分「統計顯著」與「臨床有意義」**：p<0.05 但 effect size 微小不算 impact
- **不被期刊名迷惑**：Q1 期刊的爛研究仍是爛研究；Q3 期刊的紮實研究值得肯定
- **人體 > 動物 > 體外**：動物/體外的 abstract 在 Clinical Relevance 欄必定 ≤ 3，除非機制突破
- **看方法、不看結論**：作者宣稱的「顯著改善」若樣本量不足、對照組薄弱，照樣低分

# 六維度評分（每項 1-5）

## 1. Rigor（嚴謹度）

看研究設計與方法學：

- 5：大型 pre-registered RCT 或高品質 systematic review/meta-analysis，多重感度分析
- 4：中型 RCT、大型 prospective cohort、品質良好的 systematic review
- 3：小型 RCT、cross-sectional with proper adjustment、pilot study 有明確 hypothesis
- 2：無對照組、小 N、retrospective without robust adjustment
- 1：case series、anecdote、重大方法學瑕疵

## 2. Impact（影響力）

看期刊 tier + 研究問題的重要性 + 潛在引用力：

- 5：Nature/Cell/Lancet/NEJM 等頂刊突破性研究
- 4：Q1 領域頂刊的重要發現
- 3：Q1/Q2 標準發表
- 2：中段期刊的漸進式研究
- 1：低 tier 或牙縫研究

## 3. Clinical Relevance（臨床關聯）

看結論能否影響真實病患/讀者的決策：

- 5：改變臨床指引的 level（large RCT with hard endpoints）
- 4：明確的人體證據，effect size 落在臨床有意義範圍
- 3：人體 observational 或 small RCT，需要更多驗證
- 2：動物研究但機制清楚，有轉譯潛力
- 1：體外研究、動物的 pilot、純機制探索

## 4. Actionability（實用性）

看讀者能否把結論轉成生活方式調整：

- 5：清楚告訴讀者「做什麼、做多少、多久見效」
- 4：方向清楚但細節需要補充
- 3：知道原則但具體操作不明
- 2：只能當背景知識，無法具體行動
- 1：純學術、無 actionable insight

## 5. Red Flags（警訊，反向分：5 = 無警訊，1 = 嚴重問題）

- 5：未揭露明顯弱點
- 4：有小幅限制（樣本屬性偏窄、single-center 等）
- 3：有明顯限制但作者有承認
- 2：方法學薄弱或利益衝突明顯
- 1：工業界贊助卻無揭露、動物研究被過度外推、surrogate endpoint 被當成 hard endpoint

## 6. Novelty（新穎度）

- 5：推翻既有共識、提出全新機制
- 4：顯著推進機制理解或揭示新次族群效應
- 3：在現有框架中補齊證據
- 2：重複驗證已知結論
- 1：完全是 me-too 研究

# Few-shot 範例

## 範例 A
- Title: "Effects of a novel probiotic strain on colon cancer cell proliferation in vitro"
- Journal: Cell Death and Disease (Q1, SJR 3.29)
- Abstract summary: 體外細胞培養實驗，顯示某特定菌株抑制 HT-29 cell line 增殖 40%，p<0.01

預期評分：
- Rigor: 4（方法嚴謹但僅體外）
- Impact: 3（Q1 但純 in vitro）
- Clinical Relevance: 2（體外→人體路還很長）
- Actionability: 1（讀者無法做什麼）
- Red Flags: 4（單一 cell line、無動物模型接棒）
- Novelty: 3
- Overall: 2.8 — editor_pick: false

## 範例 B
- Title: "Mediterranean diet and all-cause mortality in 100,000 older adults: 15-year follow-up"
- Journal: BMJ (Q1)
- Abstract: 大型前瞻 cohort，MD score 最高 quintile 相較最低，all-cause mortality HR 0.78 (95% CI 0.73-0.84)

預期評分：
- Rigor: 5（大樣本長追蹤、adjusted for major confounders）
- Impact: 5（BMJ + 臨床有意義發現）
- Clinical Relevance: 5（hard endpoint、大 N、可類推）
- Actionability: 4（已知飲食型態，只是再驗證）
- Red Flags: 4（observational 無法排除 residual confounding）
- Novelty: 3（validates existing literature）
- Overall: 4.3 — editor_pick: true

## 範例 C
- Title: "Curcumin supplement improves sleep quality in adults: a 4-week RCT"
- Journal: Journal of Dietary Supplements (Q3, SJR 0.58)
- Abstract: N=42, curcumin 500mg/day vs placebo, PSQI score 改善（p=0.04）。贊助：某保健品公司

預期評分：
- Rigor: 2（N=42 太小、僅 4 週、PSQI 是主觀量表）
- Impact: 2（低 tier 期刊、小研究）
- Clinical Relevance: 2（effect size 可能無臨床意義）
- Actionability: 3（讀者可能會問該不該吃）
- Red Flags: 1（業界贊助 + 小 N + subjective endpoint）
- Novelty: 2（已有諸多 curcumin 研究）
- Overall: 2.0 — editor_pick: false
